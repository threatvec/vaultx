## [1.0.0] - 2026-03-30

### Added
- **40+ Security Tools** across 7 categories
- **Domain & Web**: ShadowScan, URL Scanner, WHOIS Lookup, DNS Analyzer, SSL Inspector, HTTP Headers, Web Fingerprint, Phishing Detector
- **Cyber Threat**: NightWatch, IP Reputation, CVE Search, Threat Feed
- **Network & IP**: IP Intelligence, IP Geolocation Map, Port Scanner, Network Tools, My IP Info, BGP Lookup
- **File & Analysis**: Metadata Extractor, Image EXIF, Hash Lookup, Hash Generator, QR Analyzer, Document Analyzer
- **Identity & OSINT**: Username Search, Email Breach, Phone Lookup, Wayback Viewer, Google Dorks, OSINT Dashboard
- **Password & Security**: Password Analyzer, Password Generator, Email Header Analyzer, 2FA Generator, Encoder/Decoder, Paste Monitor
- **AI Tools**: AI Assistant, AI Risk Analysis, AI Report
- **Smart Clipboard** — auto-detect IPs, URLs, emails, hashes from clipboard
- **NightWatch Daemon** — 24/7 background breach monitoring
- **Scan Me** — comprehensive self-scan with security score
- **Command Palette** — Ctrl+K quick navigation
- **Badge System** — gamification rewards for tool usage
- **Multi-language** — Turkish and English support
- **3 Themes** — Cyber Green, Electric Blue, Red Alert
- **Dual AI Mode** — Local (Ollama) + Cloud (OpenAI/Anthropic/Gemini)
- **AES-256-GCM** encrypted local database
- **History & Log System** — detailed query logs with bookmarks
- **Scheduled Scans** — automated recurring scans
- **Security Report** — weekly PDF reports
- **Network Topology Map** — visual network graph
- **Projects/Workspace** — organize scans by project
- **Notifications** — Telegram, Email/SMTP, Discord alerts
- **Proxy Support** — SOCKS5/HTTP proxy for all requests
- **Auto-update** check on startup
- **Onboarding Wizard** for first-time setup

### Technical
- Wails v2 framework (Go + React)
- SQLite with WAL mode
- Lazy-loaded page components
- Framer Motion animations
- i18next internationalization
